# Xavier Wall CYBR 4740 Final Project

The following project is an infrastructure as code example that I have assembled as part of my CYBR 4740 Course at Weber State University. While this code may not perfectly reflect a production automated infrastructure, it is a foundation for an automated infrastructure. This project runs on five self hosted Github runners which run OpenTofu and Packer commands on a VSphere Hypervisor. This project also uses a CloudFlare R2 Bucket as a backend.
